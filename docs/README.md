# CAX - Comandiux for V Language

A command line system tool implemented in V language.

## Building

```bash
v build
```

## Usage

```bash
cax.exe [command] [parameters]
```

## Features

- System control (shutdown, reboot, logoff)
- Process management
- Service/driver management
- Device management
- Window control
- File operations
- Clipboard operations
- On Screen Display (OSD)
- Message boxes
- And more...
